# 入门

## 选择你的路径

### 使用开放本体

最快的入门方式。从 GitHub 获取 Turtle 文件，加载到你的三元组存储中。

1. [浏览本体](https://github.com/iqns-org/ontologies)
2. 选择与你相关的领域（AI、法律、金融、健康等）
3. 加载到任意 RDF 存储（Apache Jena、Blazegraph、GraphDB、Stardog）
4. 使用 SPARQL 查询

无需账户。无需注册。

---

### 通过 MCP 连接 AI 代理

如果你在构建代理工作流，MCP 服务器为你的代理提供对知识图的结构化访问。代理可以直接查询概念、关系和跨框架对齐——无需 PDF 上下文窗口。

可在[商务层](/services)使用。

---

### 托管的 SPARQL 端点

不想自行运行三元组存储？托管端点让你无需基础设施即可进行查询访问。

可在[商务层](/services)使用。

---

### 构建私有知识库

使用你自己的机构知识扩展公共本体——内部政策、定制框架、专有风险分类法。保持私有，可与公共图一起查询。

可在[商务层](/services)使用。
