## ¿Por qué una colección de casos de uso dedicada?

IQ:NS es una base semántica compartida. Cada equipo utiliza esa base de manera diferente:

- los ejecutivos necesitan evidencia coherente y visibilidad de riesgos
- los equipos legales necesitan mapeos de obligaciones consistentes y auditables
- los equipos de seguridad necesitan un vocabulario para controles y modelos de amenazas
- los ingenieros necesitan requisitos consultables y conocimiento listo para agentes
- las operaciones necesitan el mismo conocimiento estructurado en flujos de trabajo y catálogos

Cada página de caso de uso explica cómo IQ:NS conecta el conocimiento institucional, los marcos y los sistemas para que su organización pueda actuar desde la misma fuente de verdad.
