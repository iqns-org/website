# Sobre IQ:NS

## O que estamos construindo

IQ:NS é um projeto de código aberto que transforma padrões fragmentados, regulamentos e conhecimento institucional em ontologias estruturadas e consultáveis. Construídas sobre padrões da web semântica (RDF, SKOS, OWL, SHACL), as ontologias fornecem aos sistemas de IA e às organizações uma linguagem compartilhada.

## Por que isso existe

A integração de IA nas organizações é fundamentalmente um **problema de linguagem**. As equipes leem os mesmos regulamentos e chegam a conclusões diferentes. Os padrões usam termos diferentes para os mesmos conceitos. O conhecimento institucional vive em documentos que as máquinas não conseguem raciocinar.

Começamos o IQ:NS para corrigir a camada estrutural — para construir o vocabulário compartilhado que permite que pessoas e sistemas trabalhem a partir da mesma fonte de verdade.

## O que acreditamos

- **Os padrões contêm insights reais** — mas ler 40 PDFs e sintetizá‑los é onde as coisas falham. Ontologias formais tornam essa síntese reutilizável.  
- **O contexto importa** — uma IA que recomenda música não é a mesma que uma IA que pontua crédito. A estrutura deve refletir isso.  
- **Aberto é melhor** — a camada semântica para IA e conhecimento institucional deve ser um bem público, não um bloqueio proprietário.  
- **Trabalhamos com metadados** — nunca tocamos nos dados do modelo, conjuntos de treinamento ou PII. As ontologias descrevem conceitos e relacionamentos.

## A comunidade

IQ:NS é mantido por contribuidores com experiência em pesquisa da web semântica, prática regulatória e integração empresarial. As ontologias principais são gratuitas e de código aberto. Contribuições, feedback e expertise de domínio são bem‑vindos.

## Camada comercial

Para equipes que precisam de bases de conhecimento privadas, endpoints SPARQL hospedados, integração com servidor MCP ou suporte prático, há uma [camada comercial](/services). Mas as próprias ontologias permanecem gratuitas.
