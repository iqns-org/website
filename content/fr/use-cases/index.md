## Pourquoi une collection de cas d'utilisation dédiée ?

IQ:NS est une base sémantique partagée. Chaque équipe utilise cette base différemment :

- les dirigeants ont besoin de preuves cohérentes et d'une visibilité sur les risques
- les équipes juridiques ont besoin de cartographies d'obligations cohérentes et auditable
- les équipes de sécurité ont besoin d'un vocabulaire pour les contrôles et les modèles de menaces
- les ingénieurs ont besoin d'exigences interrogeables et de connaissances prêtes pour les agents
- les opérations ont besoin des mêmes connaissances structurées dans les flux de travail et les catalogues

Chaque page de cas d'utilisation explique comment IQ:NS relie les connaissances institutionnelles, les cadres et les systèmes afin que votre organisation puisse agir à partir de la même source de vérité.
