# IQ:NS

[IQ:NS](https://IQNS.org) is a curated collection of semantic knowledge assets that describe how real-world domains connect to frameworks, standards, and governance models.

IQNS acts as the "skeleton" or "semantic backbone", turning raw data into interoperable intelligence by offering structure, grounding, and explainability.


## What this repository contains

- Domain summaries that explain key concepts and relationships
- Semantic assets that capture frameworks, regulations, and standards in a reusable form
- A structured layout for exploring AI, finance, health, legal, government, and more

## How to use this repository

1. Start with the domain summary that matches your area of interest.
2. Follow the links to explore the available graphs and examples.
3. Use the collection structure to compare related concepts across domains.
